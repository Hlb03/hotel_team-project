Частина 1

1.1 Що таке HTML?
HTML (Hyper Text Markup Language) є стандартною мовою розмітки, головна ціль якої - створення веб-сторінок
1.2 HTML редактори коду та IDE
HTML-код можна написати використовуючи звичайний блокнот. Дана програма встановлена за замовчуванням на кожному комп'ютері. Проте набагато краще використовувати текстові редактори та IDE.
Одна з головних переваг текстових редакторів - підвічування синтаксису, що дозволяє значно полегшити та прискорити процес створення веб-сторінок. До них можна віднести наступні програми:
  • Notepad++;
  • Sublime Text 3;
  • Brackets та ін.
IDE (Integrated development environment) - інтегроване середовище розробки, котре містить всі можливості текстових редакторів, а також компілятори/інтернепретатори, інструменти для відлагодження коду, вбудовані системи керування версіями, інструменти для полегшення розробки графічного інтнерфейсу користувача та інший функціонал. Прикладами програм даного типу є:
  • VS Code;
  • Atom;
  • IntelliJ IDEA;
  • PyCharm та ін.
1.3 HTML Основні приклади
HTML документи мають починатись з оголошення типу документу, тобто тегу <!DOCTYPE html>.
Сам HTML документ починається з парного тегу, котрий відкривається та закривається наступним чином - <html> ... </html>
HTML документ може містити заголовки різного типу, що створюються за допогою тегів <h1> </h1>, всередину якого поміщається бажаний текст заголовка. Всього інсує 6 розмірів заголовків, відповідно від h1 та до h6, де тег з індексом 1 - найбільший за розміром.
Також, можна створити параграф за допомгою тегів <p> ... </p>. Кожен новий параграф буде розпочинатись з нового рядка.
HTML документ передбачає можливість створення посиланнь та додавання малюнків до веб-сторінок. Задля створення посилання слід написати тег <a href=""> ... </a>. У якості значення параметру href слід передати бажане посилання, а поміж тегів a написати бажане відображення посилання. Щоб додати малюнок слід набрати тег <img src="">, та в якості параметра атрибута src передати розташування малюнка.
1.4 HTML елементи
Зазвичай, елементи html сторінки, складаються з початкового тега, та кінцевого (закриваючого), а бажаний вміст вставляється посеред ними. Наприклад:
<some-tag>Бажаний зміст...</some-tag>
Кожен html документ складається з трьох основих тегів:
  1. <html> ... </html> -- визначає весь документ
  2. <head> ... </head> -- всередині даного тегу розміщуються метадана про весь документ
  3. <body> ... </body> -- містить всередині все відображення документа
1.5 HTML атрибути
Кожна складова html документу може мати атрибути, метою яких є надання додаткової інформації про елемент (у розділі 1.3 було згадано атрибути href та src).
Зазвичай, кожна картинка містить декілька атрибутів, а саме:
  • src -- посилання на місце, де розташована картинка
  • alt -- альтернативне відображення у випадку, якщо картинку на було знайдено
  • width -- бажана ширина (зазначається у пікселях)
  • height -- бажана висота (зазначається у пікселях)
Приклад тегу для додавання картинки до документу:
<img src="https://www.some.picture.jpg" width="200px" height="300" alt="Not found">
Іншим розповсюдженим атрибутом є style, котрий надає певного стилістичного обрамлення (шрифт, колір, розмір, місце розташування і т.д.)
1.6 HTML параграфи
Щоб створити параграф слід оголосити парний тег <p> ... </p>, всередину якого помістити бажаний текст. Проте слід розуміти, що html не реагує на простий перенос тексту на новий рядок.
Задля розриву між рядками слід оголосити порожній тег <br>
1.7 HTML посилання
Посилання - це певного роду текст, при натисканні на який відбувається перенаправлення на інше джерело. Загальний вигляд створення посилання за допомогою html:
<a href="source-name"> ... </a>
  1. Кольорові посилання призвані відобразити інформацію, щодо попереднього відвідування даного джерела. Загалом існує три кольори посилань:
    • синє посилання -- джерело раніше не відвідувалось
    • фіолетове посилання -- джерело вже відвідувалось
    • червоне посилання -- активне посилання
  2. Створення закладок - інструмент, що допомагає пришвидшити навігацію по сторінці. Щоб створити посилання на абзац/параграф всередині документу, слід надати абзацу/параграфу якесь значення у якості атрибуту id, та потім створити звичайне посилання, де у якості значення атрибуту href передати значення #id-name. Приклад створення закладки:
    <p id="bookmark1">Some content</p>
    <a href="#bookmark1">Here is a navigation to paragraph</a>

1.9 HTML таблиці
Таблиця дозволяє певним родом структурувати дані та відобразити їх у легкому для сприйняття вигляді. Приклад:

Результати атестації
Предмет Атестація
Предмет_1 +
Предмет_2 -
  1. HTML Межі таблиці -- можливість створювати різні межі таблиць. Приклад таблиці з подвійним зовнішнім контуром:
    Продукт Кількість (кг)
    Картопля 4
    Морква 1
  2. HTML Розмір таблиці -- можливість задавати ширину/висоту як всієї таблиці, так і окремих її елементів. Приклад:
    Heading_1 Heading_2
    Data_one Data_two
    Data_three Data_four
  3. HTML Заголовки таблиці -- можливість створити заголовки до кожної колонки за допомгою тега <th>...<\th>. Приклад:
    Заголовок_1 Заголовок_2 Заголовок_3 Заголовок_4
  4. HTML Таблиці. Padding & Spacing -- можливість корегувати відступи між чарунками таблиці. Приклад:
    Понеділок Середа П'ятниця
    Відпочинок Так Ні
    Навчання Ні Так
  5. HTML Таблиці. Colspan & Rowspan -- дозволяє об'єднувати декілька чарунок в рядок/стовпець. Приклад:

    Name Purchases
    Price Amount (kilo) Sum
    Apple 20 2 40
    Potato 10 3 30
    Carrot 15 1 15
    Total 3 85
  6. HTML Стилізація таблиць -- можливість кастомізувати рядку/стовпці. Приклад:
    Mon Wed Fri
    __ __ __
    __ __ __
  7. HTML Група стовпців таблиці -- можливість об'єнувати стовпці у групи. Приклад:

1.10 HTML Списки
Списки надають можливість перелічити які-небудь аргументи. Існують три види списків у html документах: невпорядкований, впорядкований, інші списки.
  1. Впорядковані списки -- списки, аргументи в якому йдуть по порядку (за спаданням, зростанням). Існує можливість вибирання типу нумерації у впорядкованих списках. Це відбувається шляхом надання атрибуту value певних значень. Приклад впорядковоного списку з грецькою нумерацією:
    1. First member
    2. Second member
    3. Third member
  2. Невпорядковані списки -- списки, інформація в яких присутня у якості переліку. Зазвичай, всі пункти списку еквівалентні за важливістю. Приклад:
    • First place
    • Second place
    • Third place
  3. Інші списки - до них відносять список, котрий прдставлений у вигляді пари ключ-значення. Зазвичай даний тип списків використовується при створенні списку з визначеннями та їхніми значеннями. Приклад:
    WEB-застосунок
    застосунок, у якому якості клієнта виступає браузер, а сервером - вебсервер.
