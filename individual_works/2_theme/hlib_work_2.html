CSS Контур

Загальний приклад тексту, навколо якого буде відображений контур зеленого кольору.

  1. CSS Ширина контуру
    Існує чотири можливих варіантів задання товищини контуру:
    Тонка лінія - за допомогою значення thin
    Лінія середньої товщини - за допомогою значення medium
    Жирна товщина контуру - значення thick
    Можливість самому встановлювати необхідну товщину, задаючи значення у пікселях (px)
  2. CSS Колір контуру
    Можвість змінювати колір контуру:
    Задається шляхом надання атрибуту outline-color значення бажаного кольора
  3. CSS Скорочена властивість контуру
    Задання всіх бажаних параметрів контуру скорочених шляхом:
    Це досягається шляхом написання атрибуту outline та переліку бажаних значень
  4. CSS Outline Offset
    Можна створювати контуру і на певній відстані від текста:
    Це досягається за допомогою атрибуту outline-offset та надання у якості параметра розмір відступу у пікселях

CSS Текст

Приклад тексту жовтого кольору на блакитному фоні

  1. CSS Вирівнювання тексту
    Існує багато різних типів вирівнювання тексту, зокрема:
    Центрування тексту - за допомогою надання атрибуту text-align значення center
    Притискання до одного з бортів - за допомогою надання атрибуту text-align значення right/left
    Є можливість задавати певну ширину та висоту, відповідно до яких формується необхідна кількість рядків
    Можна вирівнювати останній рядок тексту:
    Для прикладу напишему тут біль-менш достатню кількість тексту, та вирівняємо останній рядок праворуч. Це досягається шляхом задання атрибуту text-align-last та надання йому значення тієї межі, за якою має бути вірівняно текст. Існує декільках можливих значень: center, right, justify та left (за замовчуванням)
    Зміна напрямку відображення тексту:
    Досягається за допомогою атрибутів direction та unicode-biddi. Цей текст відображатиметься у звортньому порядку
    Вирівнювання картинок по вертикалі:
    Ця досягається шляхом задання атрибуту vertical-align зі значеннями top, middle або bottom
  2. CSS Оформлення тексту
    Можливість додавати декоративні лінії:
    Досягається за допомогою атрибуту text-decoration-line та задання бажаного значення (можливо комбінувати одразу декілька)
    Можливість задавати колів декоративної лінії за допомогою атрибуту text-decoration-color та встановлення бажаного значення
    Можливість вибирати стиль лінії за допомогою атрибуту text-decoration-style та надання йому певних значень (у якості приклада dotted)
    Можливість задавати товщину лінії - параметр text-decoration-thickness
  3. CSS Трансформація (перетворення) тексту
    Можливість переводити весь текст у певний регістр:
    Відбувається за допомгою атрибуту text-transform та задання одного зі значень uppercase, lowercase, capitalize
  4. CSS Інтервал тексту
    Задання інтервалів різного типу:
    Початковий абзац за допомогою атрибуту text-indent
    Можливість регулювати відстань між літерами за допомогою атрибуту letter-spacing
    Можливість регулювати значення
    відступів між рядками
    Можливість задавати відстань між словами за допомогою атрибуту word-spacing
  5. CSS Тінь тексту
    До тексту також можна додавати ефект тіні:
    За допомогою атрибуту text-shadow

CSS Синтаксис

Базовий синтаксис CSS складається із селектора та блоку об'яви. У якості прикладу, ось CSS код для заголовків третього типу в цьому документі:

  h3 {
      display: flex;
      flex-direction: row;
    }

  /* Приклад коментаря у CSS */
    h3:before,
    h3:after {
      content: "";
      flex: 1 1;
      border-bottom: 2px solid #000;
      margin: auto;
    }

Даний CSS код дозволяє виводи зоголовок поміж чорної лінії посередині

CSS Селектори

Селектори, це CSS код, що застосовується до певних елементів. Існує п'ять типів селекторів:

  • Прості селектори
    • Селектори елемента - застосовуються до всіх html-елементів певного типу (наприклад до тегу <p>)
    • Селектори ідентифікатора - застосувається до елемента з певним id
    • Селектор class - задає певну стилізацію всім елементам класу (слід використати атрибут тегів під назвою class)
    • Універсальний селектора - застосовується до всіх елементів html-сторінки (декларується символом *)
    • Селектора групування - застосовується у випадку, якщо слід задати однакову стилізацію елементам декількох типів (наприклад p, h1)
  • Комбінаторні селектори

    • Селектори нащадків - відповідають всім елементам, котрі є нащадками вказаного елемента

    • Дочірній селектор - застосовується безпосередньо до першого рівня вкладеності елементів
    • Сусідні родинні селектори - застосовується до наступного у списку тегу 
      (приклад застосування на основі тегів li + pre)
    • Загальний батьківський селектора - застосовується до всіх родинних елементів

      Приклад відображення батьківського селектора

  • Селектори псевдокласів
    • Псевдо-класи та CSS-класи - застосовується певні класи до html-тегів (приклад зміни кольору при наведенні)
    • Виринаюча підказка - вспливає при наведенні на елемент
      (наведіть курсор, щоб побачити підказку)

      З'явилась підказка

    • Псевдо клас first-child - застосовується до перших елементів певного html-тегу

      (наприклад ось курсивний елемент)

    • Псевдо клас :lang - визначає спеціальні правила для різних мов (приклад з визначенням незвичайних лапок для цитати)
  • Селектори псевдоелементів
    • Псевдо елемент ::first-line - виконує стилізацію першого рядка певного html-тега

      (приклад застосування псвевдо елемента ::first-line
      лише до першого рядка)

    • Псевдо елемент ::first-letter - застосовує стилізацію лише до першої літери

      (Приклад ::first-letter)

    • Псевдо елемент ::before - застосовує бажаний код перед кожним обраним html-елементм

      (Текст попереду за допомогою псевдоелемента ::before)

    • Псевдоелемент ::marker - стилізує маркери списку. Приклад:
      1. Перший
      2. Другий
    • Псевдоелемент ::selection - дозволяє змінювати вигляд тексту при його виділенні (Видиліть цей текст)

Створення стовпців CSS

CSS надає можливість створювати різну кількість ствопців на сторінці, що створює ефект журналу/газети. Приклад:

Таблиці стилів дають змогу спростити процес створення сторінок і поліпшити їхній зовнішній вигляд. Концепція стилів подібна до ідеї стилів, яка реалізована в сучасних текстових редакторах — текст спочатку вводять, а потім форматують, користуючись стилями. Застосування стилів дає змогу вводити на сторінку потрібні тексти та інші елементи, не задумуючись над їхнім зовнішнім виглядом і розташуванням. Таблиці стилів програміст зазвичай створює окремо від html-файлу. Під час створення html-файлу він концентрує увагу на змісті сторінки, а не на її зовнішньому вигляді, а під час створення таблиці стилів — навпаки. Отже, стилі дають змогу розмежувати етапи створення html-файлу й удосконалення зовнішнього вигляду сторінки.
