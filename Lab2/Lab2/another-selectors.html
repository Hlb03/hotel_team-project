Сусідні родинні селектори:

    <!DOCTYPE html>
    <html lang="en">
        <head>
            <title> Родинні селектори </title>
            <style>
                div + p {
                    background-color: pink;
                }
            </style>
        </head>
        <body>
            <div>
                <p>Буде застосований стиль</p>
                <p>Не буде застосовано стилізацію</p>
            </div>
        </body>

Вигляд:

Буде застосваний стиль

Не буде застосовано стилізацію


Дочірні селектори:

    <!DOCTYPE html>
    <html lang="en">
        <head>
            <title> Дочірні селектори </title>
            <style>
                div > span {
                    font-family: 'Brush Script MT', cursive;
                }
            </style>
        </head>
        <body>
            <div>
                <span>Приклад застосування дочірнього селектору</span>
            </div>
        </body>

Вигляд:

Приклад застосування дочірнього селектору

Селектор атрибутів:

    <!DOCTYPE html>
    <html lang="en">
        <head>
            <title> Селектори атрибутів </title>
            <style>
                p[class] {
                    letter-spacing: 6px;
                }
            </style>
        </head>
        <body>
            <div>
                <p class="">Приклад застосування селектора атрибутів</p>
            </div>
        </body>

Вигляд:

Приклад застосування селектора атрибутів


Універсальний селектор:

    <!DOCTYPE html>
    <html lang="en">
        <head>
            <title> Універсальний селектор </title>
            <style>
                * {
                    color: purple;
                }
            </style>
        </head>
        <body>
            <div>
                <p>Буде застосований стиль</p>
                <span>Також буде застосований стиль</span>
                <i>Курсив + загальний стиль</i>
            </div>
        </body>

Вигляд:

Буде застосований стиль

Також буде застосований стиль Курсив + загальний стиль
