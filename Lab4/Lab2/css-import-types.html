Способи підключення стилів до html-сторінки

Існує декілька різних стилів підключення CSS до html-сторінки, зокрема:

Зовнішні стилі
Під'єднується зовнішній css-файл до html-документа за допомогою тегу link. Приклад коду: 
<head>
    ...
    <link rel="stylesheet" type="text/css" href="../general/css/styles.css">
    ...
</head>
Таблиці стилів документу
Декларація всіх необхідних css-елементів безпосередньо у html-документі, в тезу style. Приклад коду: 
<head>
    ...
    <style>
        ...
    </style>
</head>
Inline-стилі
Реалазація всіх необхідних стилів безпосередньо в html-тезі елемента. Приклад коду:
<p style="...">До даного тексту будуть застосовані певні стилі</p>
